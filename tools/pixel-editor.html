Gestisce un intero file .h: Icone a colori (palette/RGB) o Font bitmap (formato myFont.h). Apri/modifica/salva, niente dipendenze, stato salvato nel browser.

Click/trascina per dipingere. Tasto destro = sfondo. Alt+click = pipetta. Scorciatoie B/E/G/L/R/I.

Icone nel file

Font & glifi

Dimensioni valide per tutti i glifi (max 8 righe: ogni colonna è 1 byte). L'etichetta di ogni glifo finisce nel commento // (vuota = nessun commento).

Colori

Indicizzata: palette di #define condivisa (formato meteo.h).

RGB libero: ogni cella un colore; export uint8[H][W][3].

Testo → pixel

Font integrato 3×5 per stampare scritte dentro un'icona. Disegna col colore selezionato; i pixel vuoti non toccano lo sfondo.
